ОПИСАНИЕ

Веб-приложение для онлайн-записи на приём. Пользователь выбирает доступную дату (выходные и прошедшие дни заблокированы), свободный часовой слот (с 9:00 до 17:00), заполняет контактные данные (имя, email, телефон). 
После подтверждения запись сохраняется в базе данных. Администратор может просматривать все записи в таблице и удалять их.

Проект разработан в рамках учебной практики и демонстрирует навыки создания полноценного веб-приложения с разделением на frontend и backend, использованием REST API, 
валидацией данных и управлением состоянием.

СТЕК 

| Фронтенд | React, React Router, Material-UI (MUI), Axios, date-fns, @mui/x-date-pickers, Vite |
| Бекенд  | Node.js, Express, better-sqlite3, CORS, dotenv |
| ДБ | SQLite |

Требования
- Node.js (версия 14 или выше)
- npm или yarn

Шаги

1. Клонировать репозиторий
   git clone https://github.com/your-username/appointment-booking.git
   cd appointment-booking

2.
  cd backend
  npm install

3.
  npm run dev - запускать бек и фронт
